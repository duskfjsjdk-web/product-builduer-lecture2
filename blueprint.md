
# Color Palette Generator

## Overview

This is a simple web application that allows users to generate random color palettes. The application is built using HTML, CSS, and JavaScript, and it follows modern web development best practices.

## Features

*   Generate random color palettes with a single click.
*   Display 5 colors per palette.
*   Show the hex code for each color.
*   Copy the hex code to the clipboard when a color is clicked.
*   Responsive design for mobile and desktop.
*   An input field to type messages and see them displayed on the page.

## File Structure

*   `index.html`: The main HTML file.
*   `style.css`: The CSS file for styling.
*   `main.js`: The JavaScript file for application logic.
*   `blueprint.md`: This file.
